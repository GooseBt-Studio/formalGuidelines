### 安装反病毒软件

安装反病毒软件的主要目的在于接管 Windows Defender 以使用一些 Windows 认为恶意但实际友善的应用程序（例如 Windows 激活工具）。

笔者建议前往[火绒安全](https://www.huorong.cn/)下载火绒安全软件个人免费版，直接使用默认值依照软件安装包的指引完成安装即可。

安装完成后，可以看到路径 ``C:\Program Files\Huorong`` 下存在一个名为 ``Sysdiag`` 的文件夹（和一个临时用于升级的文件夹），此时可执行一次快速扫描。

根据自己的偏好配置反病毒软件，以下为一些推荐的设置：

1) 建议使用密码进行保护以防篡改配置、自动化脚本模拟用户退出反病毒软件和恶意卸载等；
2) 启用压缩包扫描、扫描网络驱动器、空闲扫描、启发式查杀和 GPU 加速，病毒处理方式设置为询问我，并在处理病毒时将病毒备份至隔离区；
3) 日志记录调整至最大范围；
4) 系统加固开启全部功能；
5) 如反病毒软件允许可导入或导出配置。
