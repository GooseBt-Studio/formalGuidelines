### 偏好设置

此处仅列举部分推荐的偏好设置。

#### 一、任务栏设置

右键单击任务栏空白处，在弹出的菜单中左键单击“任务栏设置”。

- 关闭小组件。
- 处理任务栏行为：
  - 任务栏对齐方式设置为靠左；
  - 关闭自动隐藏任务栏。

#### 二、桌面个性化

右键单击桌面空白处，在弹出的菜单中左键单击个性化，进入主题，点击桌面图标设置。

在弹出的“桌面图标设置”对话框中勾选计算机和回收站，点击“应用”按钮。

右键单击桌面的回收站按钮，点击“属性”，在弹出的对话框中对所有磁盘启用“不将文件移到回收站中。移除文件后立即将其删除。”，勾选“显示删除确认对话框”，点击“应用”按钮，点击“确定”按钮。

此处怀念一下 Windows XP 年代可以对所有磁盘和后续新接入的磁盘统一使用“不将文件移到回收站中。移除文件后立即将其删除。”选项。

返回“桌面图标设置”对话框，去除回收站，点击“应用”按钮，点击“确定”按钮。

根据自己的偏好重新排列桌面图标。

#### 三、命令提示符

按下“Win + R”组合快捷键（其中“Win”键是位于键盘左侧 Ctrl 和 Alt 之间的 Windows 徽标），在弹出对话框中键入 cmd，回车。

如果是 Windows 11，请将控制台窗口设置为传统风格。

在 cmd 窗口中键入 exit 并回车，随后以相同的方式启动一个新的 cmd。

左键单击 cmd 窗口标题栏中最左边的 cmd 图标，在弹出的菜单中左键单击“默认值”。

在弹出的“控制台窗口”属性对话框中切换至“颜色”选项卡，屏幕文字设置为红 255、绿 255 和蓝 0，屏幕背景设置为 红 0、绿 0 和蓝 0，不透明度设置为 100%。

在 cmd 窗口中键入 exit 并回车，随后以相同的方式启动一个新的 cmd，观察效果。

#### 四、控制面板

在 cmd 窗口中键入 control 并回车，打开控制面板后，将右上角查看方式设置为小图标。

#### 五、文件夹选项

在控制面板中找到“文件资源管理器选项”并打开，在弹出的对话框中有三个选项卡，仅做出如下设置（更改）。

- 常规：
  - 打开文件资源管理器时打开：此电脑；
  - 浏览文件夹：在同一窗口中打开每个文件夹；
  - 按如下方式单击项目：通过双击打开项目（单击时选定）；
  - 隐私：去掉所有勾，点击“清除”按钮，点击“应用”按钮。
- 查看中的“文件和文件夹”：
  - 勾选“登录时还原上一个文件夹窗口”；
  - 取消勾选“使用复选框以选择项”；
  - 取消勾选“使用共享向导（推荐）”；
  - 取消勾选“始终显示图标，从不显示缩略图”；
  - 勾选“鼠标指向文件夹和桌面项时显示提示信息”；
  - 勾选“显示驱动器号”；
  - 勾选“显示状态栏”；
  - 取消勾选“隐藏空的驱动器”；
  - 如果是 Windows XP 及以下，取消勾选“隐藏受保护的操作系统文件（推荐）”，否则勾选该项；
  - 如果是 Windows XP 及以下，在“隐藏文件和文件夹”下选择“显示隐藏的文件、文件夹和驱动器”，否则选择“不显示隐藏的文件、文件夹和驱动器”；
  - 取消勾选“隐藏文件夹合并冲突”；
  - 取消勾选“隐藏已知文件类型的扩展名”；
  - 勾选“用彩色显示加密或压缩的 NTFS 文件”；
  - 勾选“在标题栏中显示完整路径”；
  - 勾选“在单独的进程中打开文件夹窗口”（此操作目的是为了防止一个文件夹窗口卡死后关闭该窗口导致所有文件资源管理器窗体被关闭）
  - 在“在列表视图中键入时”下选择“在视图中选中键入项”；
  - 勾选“在缩略图上显示文件图标”；
  - 勾选“在文件夹提示中显示文件大小信息”；
  - 勾选“在预览窗格中显示预览控件”，点击“应用”按钮。

点击“确定”按钮。

#### 六、用户账户控制

在控制面板中找到用户账户控制并进入，随后将等级调至最高（始终通知）；执行此操作是出于安全考虑。

关闭控制面板，在 cmd 窗口中键入 exit 并回车。

#### 七、电源和电池

单击任务栏右侧的电池图标，如果为 Windows 10 及以上系统则在弹出界面再次点击电池图标，否则在弹出的菜单中左键单击“电源和电池设置”。

在弹出的设置界面或控制面板中，仅做出如下设置（更改），此设置便于盖上盖子后程序还能继续正常运行。

- 电源模式：
  - 已接通电源：最佳能效；
  - 使用电池：最佳能效。
- 屏幕、睡眠和休眠超时（个人建议养成人走锁屏或关机等良好习惯）：
  - 已接通电源在此时间后关闭我的屏幕：从不；
  - 已接通电源使我的设备在以下时间后进入睡眠状态：从不；
  - 使用电池在此时间后关闭我的屏幕：3 小时；
  - 使用电池使我的设备在以下时间后进入睡眠状态：5 小时。
- 节能模式：在 50% 时开启。
- 盖子、电源和睡眠按钮控件：全部设置为“不执行任何操作”。

保存后，关闭上述相关窗口。

#### 八、任务管理器

右键单击任务栏空白处，在弹出的菜单中左键单击“任务管理器”。

Windows 10 及以上操作系统的任务管理器并不“专业”，因此需要执行以下操作。

1) 启动任务管理器后打开设置，将默认起始页设置为详细信息，Windows 10 及以上的详细信息对标 Windows XP 的进程，Windows 10 及以上的进程对标 Windows XP 的任务。
2) 实时更新速度设置为高。
