### 激活 Windows

推荐使用最新版 [HEU_KMS_Activator](https://github.com/zbezj/HEU_KMS_Activator/releases) 进行激活，该工具会判断当前操作系统的激活状态，并对未激活的操作系统优先选择最优方案进行激活。

下载打开后，直接在智能激活界面点击“开始”即可；激活过程需要网络连接，成功激活后 Windows 右下角将不再显示“此 Windows 未激活”等字样。

参阅：网页 [https://goosebt.com:9001/home.html](https://goosebt.com:9001/home.html) 的安装包目录下的系统安装包中有一个 Windows 激活工具合集。
