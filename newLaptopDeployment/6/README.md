### 登录微软和谷歌等账号

点击开始菜单按钮，登录 Microsoft 账号，并部署通行密钥。

打开 Microsoft Edge，登录 Microsoft 账号，启用同步，将配置同步到该设备，并自动将以后在此设备做出的配置更改自动上传并同步到其他设备。

前往 [FlClash 的 Github 发布页面](https://github.com/chen08209/FlClash/releases)下载最新版的 FlClash（或从其它已支持科学上网的设备下载安装包后导入），安装 FlClash。

前往 [https://edu.360buyimg.men/app/dashboard](https://edu.360buyimg.men/app/dashboard) 获取订阅链接（可购买 1 元/月的订阅），并通过 URL 导入 FlClash。

在 FlClash 做以下部署：

1) 在仪表盘页面启用系统代理和虚拟网卡，出站模式选择规则，如 FlClash 已启动，点击悬浮的暂停按钮；
2) 切换至工具页面，在基本配置的常规中启用 IPv6，在基本配置的 DNS 中启用 IPv6；
3) 在工具页面的应用程序中将所有设置启用；
4) 切换至代理页面，点击节点选择的自动选择，点击右下角悬浮图标，等待 10 秒左右；
5) 切换至仪表盘页面，点击悬浮的播放按钮。

打开浏览器，在地址栏中键入 [https://mail.google.com/mail/u/0/#inbox](https://mail.google.com/mail/u/0/#inbox) 直接尝试访问谷歌邮箱。

将需要登录的谷歌账号进行登录，并部署通行密钥。

将需要登录的 [GitHub 账号](https://github.com)进行登录，并部署通行密钥。

设置 Windows 密码和 PIN。
